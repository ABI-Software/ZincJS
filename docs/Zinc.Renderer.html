Class: Renderer

Zinc.Renderer(containerIn) → {Zinc.Renderer}

new Renderer(containerIn) → {Zinc.Renderer}

Create a Zinc 3D renderer in the container provided. The primary function of a Zinc 3D renderer is to display the current scene (@link Zinc.Scene} set to the renderer and each scene may contain as many geometries, glyphset and other primitives as the system can support. Zinc.Renderer also allows additional scenes to be displayed.

containerIn Object Container to create the renderer on.
