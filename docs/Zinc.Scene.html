Class: Scene

Zinc.Scene(containerIn) → {Zinc.Scene}

new Scene(containerIn) → {Zinc.Scene}

A Zinc.Scene contains Zinc.Geometry, Zinc.Glyphset and ZincCameraControls which controls the viewport and additional features. It is the main object used for controlling what is and what is not displayed on the renderer.
Parameters:
Name Type Description
